Color Palette & Properties

Colors are fundamental to web design. CSS provides multiple ways to specify colors using different formats and properties.

Color Properties
color: #3b82f6; /* Text color */
background-color: #f8fafc; /* Background color */
border-color: #e5e7eb; /* Border color */
box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1); /* Shadow with alpha */
CSS provides multiple color properties for different elements

Typography & Fonts

Typography controls how text appears on your website. CSS provides extensive control over fonts, sizes, spacing, and text styling.

Typography Properties
font-family: 'Arial', sans-serif; /* Font family */
font-size: 1.2rem; /* Font size */
font-weight: 600; /* Font weight */
line-height: 1.6; /* Line spacing */
text-align: center; /* Text alignment */
text-decoration: underline; /* Text decoration */
Typography properties control text appearance and readability

ðŸ“‹ Font Properties

  • font-family: Specifies the font family
  • font-size: Controls text size (px, em, rem, %)
  • font-weight: Controls boldness (100-900, normal, bold)
  • font-style: Controls italic (normal, italic, oblique)
  • line-height: Controls line spacing

Background Properties

Background properties control the background appearance of elements, including colors, images, gradients, and positioning.

Background Properties
background-color: #f8fafc; /* Solid color */
background-image: url('image.jpg'); /* Background image */
background-size: cover; /* Image sizing */
background-position: center; /* Image position */
background-repeat: no-repeat; /* Image repetition */
Background properties create visual depth and interest

ðŸ“‹ Background Types

  • Solid Colors: Single color backgrounds
  • Gradients: Linear and radial gradients
  • Images: Background images with positioning
  • Patterns: Repeating background patterns

Border Properties

Borders define the edges of elements and can be styled with different colors, widths, and styles.

Border Properties
border: 2px solid #3b82f6; /* All sides */
border-width: 1px 2px 3px 4px; /* Individual sides */
border-style: dashed; /* Border style */
border-color: #10b981; /* Border color */
border-radius: 8px; /* Rounded corners */
Borders add definition and visual separation

ðŸ“‹ Border Styles

  • solid: A solid line
  • dashed: A dashed line
  • dotted: A dotted line
  • double: A double line
  • groove/ridge: 3D effects
