Color Combinations

Learn about good and bad color combinations:

Good Combination
Blue background with white text
background-color: #3b82f6; /* Blue */ color: white; /* White text */

High contrast = Good readability

Bad Combination
Red background with red text
background-color: #ef4444; /* Red */ color: #ef4444; /* Red text */

Low contrast = Poor readability
