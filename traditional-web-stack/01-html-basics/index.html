HTML Structure Overview

HTML (HyperText Markup Language) is the foundation of web pages. It uses tags to structure content and create the visual elements you see on websites.

Basic HTML Document Structure
<!DOCTYPE html>
<html lang="en">
<head>
<title>Page Title</title>
</head>
<body>
<h1>Main Heading</h1>
<p>Paragraph content</p>
</body>
</html>
Every HTML document follows this basic structure

CSS Styling

  • Controls visual appearance

  • Layout and positioning
  • Makes HTML look beautiful

Heading Elements (h1 - h6)

Headings create hierarchy and structure in your content. They help users and search engines understand your page organization.

Heading Hierarchy
<h1>Main Page Title</h1>
<h2>Section Heading</h2>
<h3>Subsection</h3>
<h3>Another Subsection</h3>
<h2>Another Section</h2>
Use headings to create a logical content hierarchy
<!-- Visual representation of heading sizes --> <h1>This is H1 - Largest</h1> <h2>This is H2 - Large</h2> <h3>This is H3 - Medium</h3> <h4>This is H4 - Small</h4> <h5>This is H5 - Smaller</h5> <h6>This is H6 - Smallest</h6>

Text Elements

Text elements are the building blocks of content. They help organize and structure your text.

Paragraph and Text Elements
<p>This is a paragraph with regular text.</p>
<p>This paragraph has <strong>bold text</strong> and <em>italic text</em>.</p>
<p>You can also use <span>span elements</span> for inline styling.</p>
Text elements help organize and style your content

List Elements

Lists help organize information in structured ways. There are two main types: ordered (numbered) and unordered (bulleted).

List Structure
<ul> <!-- Unordered list (bullets) -->
<li>First item</li>
<li>Second item</li>
<li>Third item</li>
</ul>

<ol> <!-- Ordered list (numbers) -->
<li>First step</li>
<li>Second step</li>
<li>Third step</li>
</ol>
Lists organize information in structured ways

Form Elements

Forms collect information from users. They contain various input elements for different types of data.

Form Structure
<form action="/submit" method="POST">
<label for="name">Name:</label>
<input type="text" id="name" name="name">
<label for="email">Email:</label>
<input type="email" id="email" name="email">
<button type="submit">Submit</button>
</form>
Forms group related input elements together
