JavaScript Event Handling

Learn JavaScript event handling, event types, and interactive programming

Event Handling Fundamentals

JavaScript events allow you to create interactive web pages that respond to user actions.

Event Handling
// HTML
<button id="myButton">Click Me</button>

// JavaScript
const button = document.getElementById('myButton');
button.addEventListener('click', function() {
  alert('Button clicked!');
});
Events connect user actions to JavaScript functions

Advanced Event Types

Advanced event handling including form events, keyboard events, and custom event handling.

Advanced Events
// Form events
form.addEventListener('submit', function(e) {
  e.preventDefault(); // Prevent default behavior
  console.log('Form submitted');
});

// Keyboard events
document.addEventListener('keydown', function(e) {
  if (e.key === 'Enter') {
    console.log('Enter pressed');
  }
});
Advanced events provide more control over user interactions

ðŸ“‹ Event Object Properties

  • event.target: Element that triggered the event
  • event.type: Type of event (click, keydown, etc.)
  • event.preventDefault(): Prevent default behavior
  • event.stopPropagation(): Stop event bubbling
