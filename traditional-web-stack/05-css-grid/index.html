CSS Grid Basics

Basic grid setup with visual grid lines and interactive examples

Basic Grid

CSS Grid creates a two-dimensional layout system with rows and columns.

grid-template-columns: 1fr 1fr 1fr
Item 1
Item 2
Item 3
Item 4
Item 5
Item 6
Three equal columns with automatic rows
/* Basic grid setup */ .basic-grid { display: grid; grid-template-columns: 1fr 1fr 1fr; /* Three equal columns */ grid-gap: 1rem; /* Space between items */ }

Different Column Sizes

You can create columns of different sizes using fractional units.

grid-template-columns: 2fr 1fr 1fr
Wide Column (2fr)
Narrow (1fr)
Narrow (1fr)
First column is twice as wide as the others
/* Different column sizes */ .size-grid { display: grid; grid-template-columns: 2fr 1fr 1fr; /* 2:1:1 ratio */ grid-gap: 1rem; }

Auto-fit Grid

Auto-fit creates responsive grids that adapt to available space.

grid-template-columns: repeat(auto-fit, minmax(150px, 1fr))
Auto 1
Auto 2
Auto 3
Auto 4
Auto 5
Auto 6
Columns automatically fit based on available space
/* Auto-fit responsive grid */ .auto-fit-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(150px, 1fr)); grid-gap: 1rem; }

Grid with Rows

You can control both rows and columns in CSS Grid.

grid-template-rows: 100px 100px
Row 1, Col 1
Row 1, Col 2
Row 1, Col 3
Row 2, Col 1
Row 2, Col 2
Row 2, Col 3
Two rows with fixed height of 100px each
/* Grid with rows */ .row-grid { display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: 100px 100px; /* Fixed row heights */ grid-gap: 1rem; }

Grid Gap

Control the spacing between grid items with grid-gap.

grid-gap: 2rem
Item 1
Item 2
Item 3
Item 4
Item 5
Item 6
Larger gap between grid items
/* Grid with gap */ .gap-grid { display: grid; grid-template-columns: 1fr 1fr 1fr; grid-gap: 2rem; /* Space between items */ }

Column Spanning

Items can span multiple columns using grid-column.

grid-column: span 2, span 3
Spans 2 Columns
Normal
Spans 3 Columns
Items can span multiple columns
/* Column spanning */ .span-grid { display: grid; grid-template-columns: 1fr 1fr 1fr; grid-gap: 1rem; } .span-2 { grid-column: span 2; /* Spans 2 columns */ } .span-3 { grid-column: span 3; /* Spans 3 columns */ }

Row Spanning

Items can also span multiple rows using grid-row.

grid-row: span 2, span 3
Spans 2 Rows
Normal
Normal
Spans 3 Rows
Normal
Items can span multiple rows
/* Row spanning */ .row-span-grid { display: grid; grid-template-columns: 1fr 1fr 1fr; grid-template-rows: 100px 100px 100px; grid-gap: 1rem; } .span-2 { grid-row: span 2; /* Spans 2 rows */ } .span-3 { grid-row: span 3; /* Spans 3 rows */ }
