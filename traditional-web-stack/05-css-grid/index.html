CSS Grid Layout

Learn CSS Grid for complex two-dimensional layouts

Grid Fundamentals

CSS Grid is a two-dimensional layout system that makes it easy to create complex layouts with rows and columns.

Grid Container
display: grid; /* Creates grid container */
grid-template-columns: 1fr 2fr 1fr; /* Column sizes */
grid-template-rows: auto 1fr auto; /* Row sizes */
gap: 1rem; /* Space between items */
grid-template-areas: "header header header" "sidebar main aside"; /* Named areas */
Grid provides powerful two-dimensional layout control

Grid Areas & Positioning

Grid areas allow you to create named regions in your grid and position items using semantic names.

Grid Areas
grid-template-areas:
  "header header header"
  "sidebar main aside"
  "footer footer footer";

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.aside { grid-area: aside; }
.footer { grid-area: footer; }
Grid areas make layouts more semantic and maintainable

ðŸ“‹ Grid Item Properties

  • grid-area: Assign item to named area
  • grid-column: Position in columns
  • grid-row: Position in rows
  • grid-column-start/end: Column boundaries
  • grid-row-start/end: Row boundaries
