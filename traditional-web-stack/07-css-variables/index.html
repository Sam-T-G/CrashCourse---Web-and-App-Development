CSS Custom Properties (Variables)

Learn CSS custom properties for maintainable and dynamic styling

CSS Variables Fundamentals

CSS custom properties (variables) allow you to store values that can be reused throughout your stylesheet.

CSS Variables
:root {
  --primary-color: #3b82f6;
  --secondary-color: #10b981;
  --font-size: 16px;
  --spacing: 1rem;
}

.button {
  background: var(--primary-color);
  font-size: var(--font-size);
  padding: var(--spacing);
}
Variables make styles more maintainable and consistent

Advanced CSS Variables

Advanced techniques with CSS variables including fallbacks, calculations, and JavaScript integration.

Advanced Variable Usage
:root {
  --base-size: 16px;
  --scale: 1.2;
  --h1-size: calc(var(--base-size) * var(--scale) * var(--scale) * var(--scale));
}

.element {
  font-size: var(--h1-size, 2rem); /* With fallback */
  color: var(--text-color, #333); /* Fallback value */
}
Variables can be used in calculations and have fallbacks

ðŸ“‹ Advanced Features

  • Fallbacks: var(--color, #333) provides default
  • Calculations: Use with calc() function
  • JavaScript: Can be modified dynamically
  • Inheritance: Variables inherit from parent
