CSS Centering & Justification

Multiple centering and justification techniques with visual examples

Text Centering

Different ways to center and align text content.

Text Alignment
text-align: left
This text is aligned to the left side of the container.
Text starts from the left edge
Text Centering
text-align: center
This text is centered in the container.
Text is centered horizontally
Text Right Alignment
text-align: right
This text is aligned to the right side of the container.
Text ends at the right edge
Live Preview
/* Text alignment properties */ text-align: left; /* Left alignment */ text-align: center; /* Center alignment */ text-align: right; /* Right alignment */

Block Centering

Centering block-level elements using margin auto.

margin: 0 auto; width: 300px;
This block is centered using margin auto. It has a fixed width and is centered horizontally.
Block elements can be centered with margin auto
/* Block centering */ margin: 0 auto; /* Centers the block */ width: 300px; /* Fixed width required */

Flexbox Centering

Flexbox provides powerful centering and justification options.

Center (Default)
justify-content: center; align-items: center
Centered
Items are centered both horizontally and vertically
Flex Start
justify-content: flex-start
Start
Items are aligned to the start
Flex End
justify-content: flex-end
End
Items are aligned to the end
Space Between
justify-content: space-between
Left
Right
Items are spaced with space between them
Space Around
justify-content: space-around
Around 1
Around 2
Items are spaced with space around them
Space Evenly
justify-content: space-evenly
Even 1
Even 2
Items are spaced evenly
Live Preview
/* Flexbox centering and justification */ display: flex; justify-content: center; /* Horizontal alignment */ align-items: center; /* Vertical alignment */ justify-content: space-between; /* Space between items */ justify-content: space-around; /* Space around items */ justify-content: space-evenly; /* Even spacing */

Grid Centering

CSS Grid provides another way to center and align content.

Grid Center
place-items: center
Grid Centered
Items are centered in the grid
Grid Start
place-items: start
Grid Start
Items are aligned to the start
Grid End
place-items: end
Grid End
Items are aligned to the end
Live Preview
/* Grid centering */ display: grid; place-items: center; /* Centers items in grid */ place-items: start; /* Aligns to start */ place-items: end; /* Aligns to end */

Viewport Units

Using viewport units for responsive centering and sizing.

height: 50vh (50% of viewport height)
50% Viewport Height
This box is 50% of the viewport height
width: 50vw (50% of viewport width)
50% Viewport Width
This box is 50% of the viewport width
/* Viewport units */ height: 50vh; /* 50% of viewport height */ width: 50vw; /* 50% of viewport width */ height: 100vh; /* Full viewport height */ width: 100vw; /* Full viewport width */
