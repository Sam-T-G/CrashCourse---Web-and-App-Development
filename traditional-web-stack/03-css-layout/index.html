Box Model Fundamentals

The CSS box model describes how elements are rendered on the page, including content, padding, border, and margin.

Box Model Components
Content /* The actual content */
Padding /* Space inside the element */
Border /* Border around the element */
Margin /* Space outside the element */

Centering Techniques

Centering elements is one of the most common CSS challenges. There are multiple techniques depending on what you're centering.

Centering Methods
text-align: center; /* Inline elements */
margin: 0 auto; /* Block elements */
display: flex; justify-content: center; /* Flexbox */
position: absolute; left: 50%; transform: translateX(-50%); /* Absolute positioning */
Different centering methods for different scenarios

ðŸ“‹ Centering Scenarios

  • Inline elements: Use text-align: center
  • Block elements: Use margin: 0 auto
  • Flexbox: Use justify-content and align-items
  • Grid: Use place-items: center
  • Absolute positioning: Use transform

CSS Positioning

CSS positioning allows you to control where elements appear on the page and how they interact with other elements.

Position Values
position: static; /* Default, normal flow */
position: relative; /* Relative to normal position */
position: absolute; /* Relative to nearest positioned parent */
position: fixed; /* Relative to viewport */
position: sticky; /* Sticks when scrolling */
Positioning controls element placement and stacking

ðŸ“‹ Positioning Properties

  • top, right, bottom, left: Offset from positioned edge
  • z-index: Controls stacking order
  • position: Sets positioning context
