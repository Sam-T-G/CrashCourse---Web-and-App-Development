CSS Flexbox Layout

Master CSS Flexbox for flexible and responsive layouts

Flexbox Fundamentals

Flexbox is a powerful layout method that makes it easy to align and distribute space among items in a container.

Flexbox Container
display: flex; /* Creates flex container */
flex-direction: row; /* Main axis direction */
justify-content: center; /* Main axis alignment */
align-items: center; /* Cross axis alignment */
flex-wrap: wrap; /* Allow wrapping */
Flexbox provides powerful alignment and distribution tools

Advanced Flexbox

Advanced flexbox techniques for complex layouts including flex items properties and responsive design.

Flex Item Properties
flex-grow: 1; /* Grow to fill space */
flex-shrink: 0; /* Don't shrink */
flex-basis: 200px; /* Initial size */
flex: 1 0 200px; /* Shorthand */
align-self: flex-end; /* Override align-items */
Flex items can have individual properties

ðŸ“‹ Flex Item Properties

  • flex-grow: How much item should grow
  • flex-shrink: How much item should shrink
  • flex-basis: Initial size before growing/shrinking
  • align-self: Override container's align-items
  • order: Change visual order
