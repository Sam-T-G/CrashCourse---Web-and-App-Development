Learn to manipulate the Document Object Model with JavaScript

DOM Fundamentals

The Document Object Model (DOM) represents the structure of HTML documents and allows JavaScript to interact with elements.

DOM Selection
// Select elements
const element = document.getElementById('myId');
const elements = document.querySelectorAll('.myClass');
const firstElement = document.querySelector('.myClass');

// Modify content
element.textContent = 'New text';
element.innerHTML = '<strong>Bold text</strong>';
DOM methods allow you to select and modify elements

DOM and Events

Combining DOM manipulation with event handling to create dynamic, interactive web pages.

DOM + Events
// Create and add elements
const newElement = document.createElement('div');
newElement.textContent = 'New element';
newElement.className = 'my-class';
document.body.appendChild(newElement);

// Remove elements
element.remove();

// Modify styles
element.style.backgroundColor = 'red';
DOM manipulation combined with events creates dynamic interfaces

  • style: Modify inline styles
